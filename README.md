 🤖 Jarvis AI Assistant

An AI-powered desktop voice assistant built with Python that can perform various tasks using voice commands. Jarvis can answer questions using AI, open websites, play music, fetch news headlines, and automate everyday desktop operations.



 ## Features

-  Voice Command Recognition
-  AI-Powered Conversations (Google Gemini API)
-  Text-to-Speech Responses
-  Open Websites (Google, YouTube, Facebook, etc.)
-  Play Music
-  Latest News Headlines
-  Desktop Automation
-  Fast Command Execution



##  Tech Stack

- Python 3
- SpeechRecognition
- PyAudio
- pyttsx3
- Google Gemini API
- Requests
- Webbrowser
- PyAutoGUI
